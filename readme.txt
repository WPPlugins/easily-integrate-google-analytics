This plugin allows you to add Google Analytics of your wordpress blog by hooking into wp_head.

== Description ==

Easily Integrate Google Analytics by AMKapps is a simple light weight plugin that lets you add  Google Analytics to your WordPress site without having to get your hands dirty in code. Moreover, the simple interface gives you one place where you can add Google Analytics rather than dealing with dozens of plugins.

If you enjoy using this plugin and want to learn more about how to use WordPress then I would suggest visiting [Amkapps](http://amitytechnologies.in "Amity Technologies").

== Installation ==

1. Upload `easily-integrate-google-analytics` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin (go to the Settings/Google Analytics)
4. Enter Google Analytics ID
5. Save Changes
